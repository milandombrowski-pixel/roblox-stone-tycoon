# Stone Tycoon - Roblox Game

A kid-friendly stone processing factory tycoon game built in Roblox.

## Features

- **Mining System**: Click or auto-mine stones from the quarry
- **Machine System**: Process and combine stones into new materials
- **Upgrade System**: Unlock faster production and new machines
- **Currency System**: Earn money by selling processed materials
- **Expandable Map**: Unlock new areas as you progress
- **Leaderboard**: Track player wealth and progress

## Game Structure

### Map Zones
1. **Starter Area** - Stone mine and basic machines
2. **Factory Area** - Processing machines and combinations
3. **Shop Zone** - Upgrades and machine purchases
4. **Expansion Zones** - Locked areas unlocked through progression

### Core Systems
- **LeaderStats**: Player money and statistics
- **Mining**: Click-to-mine and auto-mining
- **Machines**: Process stones into materials
- **Upgrades**: Improve production speed and output
- **UI**: Display money, upgrades, and player stats

## Quick Start

1. Open Roblox Studio
2. Create a new place
3. Copy all scripts from `src/` folders to their respective locations
4. Build the map following MapStructure.md
5. Test in-game

## Customization

Edit config files to adjust:
- Machine prices and production rates
- Upgrade costs and benefits
- Mining speeds
- Material values
